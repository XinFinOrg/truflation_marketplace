# TFI Marketplace

create .env files with

MAINNET_PRIVATE_KEY
TESTNET_PRIVATE_KEY
INFURA_API_KEY
ETHERSCAN_API_KEY



To run
------
yarn install
yarn build
yarn test


To deploy
---------

edit scripts/config.js
npx hardhat run (deploy script) --network network

change password in chainnode

create bridges

test -> http://api-adapter:8080/
main

Run

docker-compose up
